MacOS users should use the Option (or ‚å•) key in place of the alt key where shown in the keyboard shortcuts.

  • alt + k - Keyboard shortcut list (this dialog)
  • alt + t - Show Table of Contents
  • alt + h - Show Highlights and Notes dialog
  • alt + , - Show Display settings
  • alt + . - Show Read-aloud settings
  • alt + a - Start/stop read-aloud
  • space - Pause read-aloud (while reading)
  • alt + f - Focus search input (if one is available)
  • alt + r - Focus the reader document
  • alt + pageUp - Previous chapter/section
  • alt + pageDown - Next chapter/section
  • alt + l - Go to the Library page

When text is selected within a reading:

  • alt + d - Word lookup
  • alt + h - Create highlight from selection
  • alt + s - Transform content in selection
  • alt + a - Read text in selection
