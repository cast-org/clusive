About PDF files in Clusive

This reading is a PDF. PDF files are not flexible like other readings in Clusive; so many Clusive features will not work with this reading.

Clusive features that work with this PDF are:

  • Page navigation
  • Your thoughts
  • Marking a reading with a star

There are also two buttons in the left toolbar, to zoom in and zoom out.

Clusive features that do not work with this PDF are:

  • Text size
  • Line spacing
  • Letter spacing
  • Font choice
  • Color choice
  • Read aloud
  • Transform (including Translate, Simplify text, and Show pictures)
  • Highlight & notes
  • Dictionary lookup
