About PDF files in Clusive

{% include "shared/partial/dialog_tts.html" %}

You are currently viewing a reading that is in the PDF format.

Unfortunately, PDF files are not flexible like other files in Clusive; many of the settings and supports will not work. You cannot use:

  • Read-aloud
  • Highlighting
  • Dictionary lookup
  • Text transformations
  • Text size
  • Line spacing
  • Letter spacing
  • Font preference
  • Color preferences do not apply to the PDF
  • Only "turn pages" navigation is supported

However, other features do work:

  • Page navigation
  • Your thoughts
  • Star reading

And there are two extra buttons in the left toolbar, to zoom in and zoom out.
