How to upload readings

Content in EPUB, Word, or PDF formats can be uploaded directly to Clusive.

✓ EPUB

EPUB is a common format for electronic books. You can find EPUBs at Standard eBooks, Tar Heel Reader, or other sites. You can also create EPUBs from Word documents with the free DAISY WordToEpub Converter (Windows only), or from open resources using OER Commons OpenAuthor.

✓ Microsoft Word

Make sure it is saved in the latest Word format (.docx).

If you have a Google Doc, use the File - Download - Microsoft Word menu item to get a .docx file to upload.

PDF

You can upload a PDF file directly. Note, however, that many of Clusive's adaptive features are not supported in PDF readings. For full access to all Clusive has to offer, consider converting your PDF into an EPUB or Word document. Just 5 minutes of effort can make a huge difference in the reading experience!

If you have the desktop version of Microsoft Word, open your PDF with it for a one-step conversion to a flexible format.

Add from Bookshare

If you have a Bookshare account, you can search for and add Bookshare books right here in Clusive. {% if request.clusive_user.can_upload %} Go to My Account and click Connect to a Bookshare account to get started.
